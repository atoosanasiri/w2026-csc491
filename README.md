# CSC491/2600 - Capstone Design Course

This repository contains course materials for the University of Toronto's capstone design course offered through the Department of Computer Science Innovation Lab (DCSIL).

## Course Description

Students develop a software application implementing startup business ideas from the corequisite course CSC454/2526. This hands-on capstone course provides an opportunity to apply theoretical knowledge to real-world product development.

## Course Website

Visit the official course website: [https://csc491.dcsil.ca/](https://csc491.dcsil.ca/)

## Course Objectives

Students are expected to:

- Define their own problem scope within an assigned theme
- Create a comprehensive development plan
- Produce a functioning minimum viable product (MVP)
- Present work through multiple formats: written, oral, visual, and digital

## Key Learning Outcomes

Through this course, students develop competency in:

- Writing code and learning new technologies independently
- Architecting and designing software systems
- Managing and executing software projects
- Collaborating across disciplines (business, finance, etc.)
- Establishing a robust Software Development Lifecycle with continuous integration, testing, and deployment

## Course Requirements

### Prerequisites

- Co-requisite enrollment in CSC454/2527 or prior completion
- Minimum 4 CSC 300+ level courses plus one additional 300+ level course
- Prior software development experience (web applications preferred)
- Good academic standing
- Successful application through DCSIL website

### Materials

- Laptop with functional development environment
- GitHub account

## Evaluation Criteria

Assessment focuses on students' ability to:

- Actively participate and communicate with team members
- Substantially contribute code to the project
- Adapt and self-teach required technologies
- Use GitHub for documentation and version control

## Course Format

Weekly three-hour sessions (Thursdays, 6-9 PM EST) combining workshops, lectures, discussions, and project work time. Format alternates between in-person (DCSIL Lab, Gerstein Library) and online (Zoom).

**Instructor:** Victoria Bukta, Product Manager at Databricks and U of T alumna

## Building and Previewing Site Locally

This site uses Jekyll with the Just the Docs theme.

2. Run the local server: `bundle exec jekyll serve`
3. Visit `localhost:4000` in your browser
